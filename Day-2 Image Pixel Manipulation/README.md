## Image Processing with NumPy Array Operations

Demonstrates low-level image manipulation treating images as NumPy arrays.

**Operations implemented:**
- Loaded image as a raw NumPy array via `matplotlib.image.imread`
- Grayscale conversion using the ITU-R luminosity formula
- Brightness adjustment via scalar multiplication + clipping
- Side-by-side comparison with Matplotlib subplots

**Skills demonstrated:** NumPy array slicing, broadcasting, multi-channel
array manipulation, image processing fundamentals

**Run:** `python image_processing.ipynb` (place any .jpg in the same folder)
