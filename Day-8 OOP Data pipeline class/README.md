# 🧩 OOP Data Pipeline Class

## 📌 Overview
A reusable Python class for end-to-end CSV data processing — including loading, cleaning, describing, filtering, and exporting data.

This project is built using **Object-Oriented Programming (OOP)** principles and mimics real-world **Machine Learning engineering pipelines**, making it modular, scalable, and production-ready.

---

## 🚀 Features

- 🔗 **Method Chaining**  
  Enables clean, readable one-line pipeline execution  

- 🧹 **Automatic Data Cleaning**  
  - Removes duplicate rows  
  - Handles missing values using median imputation  

- 🏷️ **Type Hints & Docstrings**  
  - Improves code readability and maintainability  
  - Follows professional coding standards  

- ⚠️ **Exception Handling**  
  - Handles `FileNotFoundError` and `ValueError`  
  - Provides clear, descriptive error messages
